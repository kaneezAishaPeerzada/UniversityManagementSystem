A Python-based University Management System developed using object-oriented programming and inheritance concepts. 
The system models real-world entities such as Person, Student, Faculty, and Staff, each with their own attributes and behaviors.
Students can enroll in courses, faculty members can be assigned to teach subjects, and staff can be assigned administrative tasks.
This project demonstrates the practical use of inheritance, method overriding, and class hierarchies in Python,
making it a useful example for learning OOP principles in academic management scenarios.
